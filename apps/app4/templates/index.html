Model 4: Draw a molecule or drug by name

Put a drug or molecule name (e.g. aspirin) in the box, press Predict to return an image of the molecule.

The tool uses PubChemPy to search PubChem for the name and retrieves the first associated SMILES structure.

It then creates a molecule from the retrieved SMILES using Rdkit and renders it on the result page.
